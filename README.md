# Valeria Villanueva's Portfolio

A responsive retro-millennial portfolio prototype built with plain HTML, CSS, and JavaScript.

## Preview locally
Open `index.html` in Chrome, Safari, or Firefox.

## Included
- Responsive homepage
- Reusable retro window and pixel-button styles
- Hero and personal photography
- Featured project placeholders
- Human-driven innovation positioning
- Email, LinkedIn, Instagram, and résumé PDF links

## Before publishing
1. Replace placeholder project visuals and `#` case-study links.
2. The résumé PDF is included at `assets/valeria-villanueva-resume-2026.pdf`. Replace that file when updating the résumé.
3. Review all copy and image alt text.
4. Deploy the folder using Netlify, Vercel, or GitHub Pages.
