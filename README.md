# Valeria Villanueva Portfolio

A responsive retro-millennial portfolio prototype built with plain HTML, CSS, and JavaScript.

## Preview locally
Open `index.html` in Chrome, Safari, or Firefox.

## Included
- Responsive homepage
- Reusable retro window and pixel-button styles
- Interactive featured project switcher
- Case Study 01: The River of Life Church
- Case Study 02: Robotics Program System Design
- Case Study 03: Monster Lab / Learning Experience Design + Program Alignment
- Hero and personal photography
- Human-driven innovation positioning
- Email, LinkedIn, Instagram, and résumé PDF links

## Key files
- `index.html` — page structure
- `styles.css` — visual design and responsive layout
- `script.js` — project tabs and slider behavior
- `assets/valeria-villanueva-resume-2026.pdf` — résumé PDF
- `assets/vex-team-guidelines-expectations.pdf` — Robotics case-study supporting artifact
- `assets/student-engineering-journal-push-back.pdf` — Robotics student engineering journal sample
- `assets/student-engineering-journal-vex-v5.pdf` — Robotics student engineering journal sample
- `assets/hs-robotics-unit-1.pdf` — Robotics curriculum slide deck
- `assets/demystifying-robotics-workshop.pdf` — Robotics pedagogy workshop slide deck
- `assets/kis-design-innovation-program-guide.pdf` — KIS Design & Innovation Program Guide for Case Study 03
